### Stack Operation
- $<[]  			= push
- $>    			= pop

- ;				= copy last stack bits. 

### Bitwise Operation

- |     			= or
- &     			= and
- ^     			= xor
- ~     			= bitwise complement

- %				= reverse bits

- <<[]   			= left shift
- \>>[]			= right shift

### IO Stream

- @<    			= read from input
- @>    			= write to output
